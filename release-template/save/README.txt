GamingLooper uses this folder as the default local save location.

The app can create gaminglooper-settings.json here on first launch.

In a directory release, this save folder is the initial location for:

- Project save: gaminglooper-settings.json
- Sequence files
- SE Set files

If you change the Sequence or SE Set save folder in Config > Files & Save,
GamingLooper remembers that custom folder and uses it instead.

Audio files are not copied or moved automatically. Saved data keeps file
references. Bundled starter SE files are stored in asset/se next to this save
folder.

Do not place the release folder in a write-protected location such as
Program Files. Use Documents or another user-writable folder.
